# Editor de CV 

Editor web para crear y personalizar currículums vitae para Community Managers con generación de PDF.

## Características

- ✅ **Sistema de autenticación**: Registro e inicio de sesión de usuarios
- ✅ **Editor visual**: Interfaz intuitiva para editar todas las secciones del CV
- ✅ **Avatares circulares**: 4 opciones predeterminadas + subir foto personal
- ✅ **Vista previa en tiempo real**: Cambios visibles inmediatamente
- ✅ **Generación de PDF**: Exportar CV a formato PDF listo para imprimir
- ✅ **Vista pública**: Compartir CV en línea
- ✅ **Responsive**: Funciona en dispositivos móviles y desktop
- ✅ **Sin solicitud de pago**: No pide métodos de pago al ingresar
